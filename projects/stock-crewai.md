# stock-crewai

## 项目概述

- **名称**: stock-crewai
- **类型**: 股票自动交易系统（虚拟盘）
- **版本**: v4.0 (2026-05-22)
- **状态**: 活跃开发中
- **路径**: D:\qclaw-workspace\stock-crewai

## 核心功能

1. **实时行情获取**: 新浪财经 API（35只沪深300成分股）
2. **技术指标分析**: MA/MACD/RSI/KDJ/布林带/ATR
3. **交易决策**: 综合评分 + 信号过滤
4. **风险管理**: 动态止损（基于ATR）、固定止盈
5. **自动执行**: 交易时段每10分钟运行一次
6. **通知系统**: 企业微信 Webhook 集成（待配置）

## 技术架构

### 主要模块

- `run_virtual_v4.py` - 主程序（循环模式）
- `technical_indicators.py` - 技术指标计算
- `portfolio_tracker.py` - 持仓管理
- `risk_manager.py` - 风险管理（Kelly公式、VaR）
- `wechat_notifier.py` - 企业微信通知
- `performance_report.py` - 绩效报告

### 数据源

- **实时行情**: 新浪财经 API (http://hq.sinajs.cn/)
- **历史K线**: 腾讯财经 API (https://web.ifzq.gtimg.cn/)
- **备用**: akshare (已弃用，限流严重)

## 交易策略

### 买入条件

1. 技术评分 ≥ 3 分
2. 市场状态非熊市
3. 持仓数量 < 5 只
4. 现金充足

### 卖出条件

1. 止损: 价格 < entry_price × (1 - atr_stop)
2. 止盈: 价格 > entry_price × 1.20
3. 技术信号: SELL 或 STRONG_SELL

### 持仓管理

- **最大持仓**: 5 只
- **止损方式**: 动态（基于ATR）
- **止盈方式**: 固定 20%
